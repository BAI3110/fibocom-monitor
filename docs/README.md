# Fibocom Monitor
**Только для модемов серии L8x0-GL**
*Multilanguage README*
[![en](https://img.shields.io/badge/lang-en-red.svg)](./README.EN.md)

## Описание:
* Показывает показатели сигнала от основной станции и дистанцию до неё.
* Показывает список соседних сот и список активых агрегаций модема.
* Показывает имя оператора и протокол сети.

* (Для L860) Показывает температуру модема.
