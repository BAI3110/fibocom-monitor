# Fibocom Monitor
**For L8x0-GL series modems only**  
*Multilanguage README*
[![ru](https://img.shields.io/badge/lang-ru-green.svg)](./README.md)

## Description:
* Displays signal metrics from the main cell and the distance to it.
* Shows a list of neighboring cells and the modem's active aggregations.
* Displays the operator name and network protocol.

* (For L860) Shows the modem temperature.
